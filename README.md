# FocusFlow ⚡

A fully interactive, browser-based gamified productivity system designed for the Gamified System Competition.

## 🎮 Live Demo
Open `index.html` in any modern browser. No installation required.

## 🏆 Gamification Mechanics
- **XP & Leveling** — Earn XP from tasks and focus sessions; trigger level-up animations
- **Combo System** — Chain tasks within 10 seconds for 2x–5x XP multipliers
- **Coins** — Secondary reward currency for sustained engagement
- **Achievement Badges** — Unlock badges based on real user milestones
- **Daily Streaks** — Maintain consecutive-day productivity with flame animations
- **Live Leaderboard** — Real-time competitive ranking against other users
- **Active Challenges** — Structured goals with progress tracking and completion rewards

## 🛠️ Tech Stack
- HTML5 / CSS3 / Vanilla JavaScript
- Canvas particle animations
- Responsive card-based UI
- Modular client-server architecture (frontend demo)
